### 1. 创建了 v2_get_bag.py 文件
- **接口路径**: `/v2/bag/get`
- **请求方式**: GET
- **请求参数**:

  - `box_id`: 储物箱ID（必填）

### 2. 核心功能实现

- **数据查询**: 使用 `SELECT * FROM bags_summary` 获取指定储物箱下的所有袋子记录
- **数据排序**: 按照 `sort_id` 升序排列返回结果

  "data": {

    "total_count": 数量
  }
}
